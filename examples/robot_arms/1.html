Robot Arms 1
Use the 'c' key to change the random solid arm color.
Use the 'C' key to randomly change arm segment colors.
Use the 'r' key to randomly change arm segment end colors.
Use the 'R' key to randomly change arm hinge colors.
Use the s/S keys to rotate the arm at the shoulder.
Use the e/E keys to rotate the arm at elbow 1.
Use the w/W keys to rotate the arm at wrist 1.
Use the f/F keys to rotate the arm at finger 1.
Use the q/Q keys to rotate the arm at elbow 2.
Use the z/Z keys to rotate the arm at wrist 2.
Use the d/D keys to rotate the arm at finger 2.
Use the x/X keys to translate the arm along the x-axis.
Use the y/Y keys to translate the arm along the y-axis.
Use the '=' key to reset the robot arm.
