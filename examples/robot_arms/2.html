Robot Arms 2
Use the 'c' key to change the random solid arm color.
Use the 'C' key to randomly change arm segment colors.
Use the 'r' key to randomly change arm segment end colors.
Use the 'R' key to randomly change arm hinge colors.
Use the s/S keys to rotate arm 1 at its shoulder.
Use the e/E keys to rotate arm 1 at its elbow.
Use the w/W keys to rotate arm 1 at its wrist.
Use the f/F keys to rotate arm 1 at its finger.
Use the a/A keys to rotate arm 2 at its shoulder.
Use the q/Q keys to rotate arm 2 at its elbow.
Use the z/Z keys to rotate arm 2 at its wrist.
Use the d/D keys to rotate arm 2 at its finger.
Use the x/X keys to translate the arm along the x-axis.
Use the y/Y keys to translate the arm along the y-axis.
Use the u/U keys to translate arm 2 along the x-axis.
Use the v/V keys to translate arm 2 along the y-axis.
Use the '=' key to reset the robot arms.
