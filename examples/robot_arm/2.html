Robot Arm 2
Use the 'a' key to toggle antialiasing on and off.
Use the 'g' key to toggle gamma correction on and off.
Use the 'c' key to change the random solid arm color.
Use the 'C' key to randomly change arm segment colors.
Use the 'r' key to randomly change arm segment end colors.
Use the 'R' key to randomly change arm hinge colors.
Use the s/S keys to rotate the arm at the shoulder.
Use the e/E keys to rotate the arm at the elbow.
Use the w/W keys to rotate the arm at the wrist.
Use the f/F keys to rotate the arm at the finger.
Use the '=' key to reset the robot arm.
